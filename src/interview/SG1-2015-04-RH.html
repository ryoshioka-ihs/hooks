学生時代～就職活動

学生時代から漠然と、パソコンの知識を身に付けたいな、パソコン周りに詳しくなりたいな、と思っていました。IT業界がどんどん発展しているイメージがあって、私もパソコンを使って何かをしたいな、と。


就職活動でもIT系の会社を探していて、就職情報サイトの運営会社が企画する大きなイベントに参加したり、個別の会社説明会に行ったり、会社をいくつも見ていました。IHSは大学のキャリアセンターで、先輩が入社していると紹介されて知りました。会社説明会では「社員はIT未経験者が多い」と伺って興味を持ちました。

IHSを選んだ理由

決め手は、入社前に社内の様子が分かったことです。毎月社内で行っているという交流会に参加させてもらえましたし、社員や同期との交流の機会が多く設けられていて、明るく馴染みやすい環境に惹かれました。


それまでに私が嫌だなと思った会社は、堅苦しいとか息が詰まる感じがするとか、社員との交流がないとか、接点があっても先輩への質疑の時間が少しあるだけとか、実態がよく分からない会社でした。IHSは入社前に様子を見られましたし、雰囲気が良かったので安心できました。

入社後の研修について

最初の2カ月間はITの資格を取るという目標があって、資格の勉強を通して基礎の知識を叩き込みました。その後の1カ月も、部署に配属される前の社内研修があり、新人でまとまってキッティング（PC環境構築）や手順書作成方法を学びました。4カ月目からは配属先のグループにて、上司にあたる人たちが用意してくれたカリキュラムと研修の資料に沿って実機を操作して学びました。


私の配属先は構築のチームでした。主にサーバー（サービスを提供する側のコンピューター）を作る部署です。私は未経験でしたが、まずは何種類かのサーバーを教えてもらって、上司に相談しながらいくつものサーバーを構築していくことで技術を身に付けていきました。

運用と構築の違い

運用は比較的やることが決まっていて、既存のマニュアルや手順書があるため業務が安定していることが多い中で、改善できることはしていきましょう、という業務です。構築はその時の案件によって構築するでサーバーの種類も内容も違います。例えばサーバーに新しい設定を入れることで、それがシステム全体にどういう影響が出るするのかなどを考えていく必要がもあります。どちらをやるかは、本人の志向やスキル次第ですね。

得られる経験やスキル

特に勉強になったと思う経験は、自動車部品メーカーのサーバー構築です。海外に拠点があるのでインフラの規模は大きく、サーバー構築の中で海外拠点との連携を意識する場面もあって刺激的でした。それ以外にも様々なサーバー構築の知識や経験が、現在も運用の現場で活かせています。


スキルは、技術者向けの内容となりますが、基礎的なActive Directory (アクティブディレクトリ／Windowsのサーバーに搭載されている機能)から、VDI（Virtual Desktop Infrastructure／デスクトップ仮想化技術）やSCCM（Microsoft System Center Configuration Manager／Microsoft社のサーバー関連製品）などを構築・運用したり、Azure（アジュール／クラウドのプラットフォーム）に触れる機会もあります。

IHSの魅力だと思うところ

「話をすれば聞いてくれる」、とか「相談すればアドバイスしてくれる」、という仕組みが魅力だと思います。IHSにはキャリアカウンセラーがいて、私も相談したことがあります。一定の期間、面談を組んで、直接話せないような仕事上の悩みとか、会社に対してこういうことをしてほしいという話をします。するとカウンセラーが会社に共有してくれて、対応してくれる仕組みです。


上司との面談も定期的にあります。現場のリーダーが、仕事上の不満や、周りの人間関係などの近況をいくつかの質問項目に沿って聞いてくれて、何かあれば調整してくれます。月一で社員が集まる交流会があるのもうれしいです。私は今、客先にいますが、同期とか他の部署の人と定期的に交流会で会ってそれぞれの近況を話し合っています。


外部の研修を受けさせてくれることも魅力です。会社から、いつまでに何個以上のセミナーを受講してください、と案内があって。セミナーのテーマには幅広い候補があって、自分で興味があるものを選べます。私の場合はロジカルシンキングとか、課題解決方法とかのセミナーを受けました。

先輩から就職活動中のみなさんへ

新しいことを知ろうという意欲を持って、自分から質問をしていけば、いろんなことを学べてスキルアップできる環境だと思います。IHSは積極的な人の話をきちんと聞いて、対応してくれる会社です。自分の意見もどんどん言った方がいいと思います。私は人目を気にしてしまうタイプで、いろいろと抱え込んでしまって悩んだ時期もありました。その時にキャリアコンサルタントや上司に相談したらちゃんと味方になってくれて、最終的に乗り越えることができました。それから自分から発信して行動できるようになって、働きやすくなりました。自分が苦しんで成長した経験は役に立つと思いますし、積極的な態度でなんでも相談してほしいなと思います。
